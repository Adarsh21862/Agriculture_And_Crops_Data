# Agriculture Data Analysis Dashboard

## Overview
This project focuses on analyzing agricultural data using Power BI to uncover insights related to crop trends, rainfall patterns, farmer distribution, and market pricing. The dashboard helps in understanding key factors that influence agricultural productivity and decision-making.

---

## Objectives
- Analyze crop types and their distribution
- Study the impact of rainfall on agricultural output
- Track market prices for different crops
- Understand farmer demographics and experience

---

## Features
- Interactive Power BI dashboard
- Visualization of crop trends and rainfall data
- Market price analysis across different crops
- Farmer distribution by gender and experience
- KPI indicators for better decision-making

---

## Tools & Technologies
- Power BI
- Microsoft Excel (Data Source)

---

## Data Processing
- Cleaned and transformed raw agricultural data
- Handled missing and inconsistent values
- Structured datasets for accurate reporting
- Created relationships between multiple data fields

---

## Key Insights
- Identified patterns between rainfall and crop yield
- Observed variation in market prices across crop types
- Highlighted distribution of farmers based on experience
- Analyzed pesticide usage across crops
